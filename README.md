# 점.메.추 (점심 메뉴 추천)

React로 만든 간단한 점심 메뉴 추천 웹앱입니다.  
나라별 / 재료별 / 완전 랜덤으로 점심 메뉴를 추천해 주고, 선택한 조건으로 다시 뽑기 기능도 지원합니다.

---

## ✨ 주요 기능

### 1. 모드 선택

- **나라별 추천**
  - 한식 / 중식 / 일식 / 양식 중에서 선택
- **재료별 추천**
  - 밥 / 면 / 떡 / 빵 중에서 선택
- **완전 랜덤**
  - 모든 메뉴 중에서 랜덤으로 하나 추천

### 2. 서브 카테고리 선택

- 나라별 모드일 때: 한식 / 중식 / 일식 / 양식 버튼 표시
- 재료별 모드일 때: 밥 / 면 / 떡 / 빵 버튼 표시
- 선택된 버튼은 하이라이트 스타일로 표시됩니다.

### 3. 메뉴 카드 표시

- 오늘의 점심 메뉴 제목
- 음식 사진
- 메뉴 이름
- 간단한 설명
- 태그:
  - 예: `한식 🇰🇷`, `밥 🍚` / `중식 🇨🇳`, `면 🍜` 등

메뉴 설명이 길어도, 카드 높이는 고정이고 **카드 안에서만 스크롤**이 생기도록 처리했습니다.

### 4. 다시 추천 / 처음으로

- **다시 추천**
  - 현재 모드 + 서브 선택 조건을 유지한 채, 같은 조건에서 메뉴만 다시 랜덤으로 추천
  - 조건이 아직 선택되지 않은 경우에는 비활성화(disabled) 처리
- **처음으로**
  - 모드 / 서브 / 결과를 모두 초기화

### 5. 상태 표시

- 화면 하단에 현재 상태를 텍스트로 표시
  - `현재 모드: 나라별 추천 / 재료별 추천 / 완전 랜덤 / 선택 안됨`
  - `세부 선택: korean / rice / 없음` 등

---

## 🛠 기술 스택

- **프론트엔드 프레임워크**: React
- **번들러 / 개발 서버**: Vite
- **언어**: JavaScript (ES6+)
- **스타일링**: CSS (모듈 분리: `App.css`, `ResultBox.css`)

---

## 📁 폴더 구조

대략적인 구조는 아래와 같습니다.

```bash
LUNCH_MENU/
├─ public/
├─ src/
│  ├─ assets/
│  │  └─ menus/          # 메뉴 이미지들
│  ├─ components/
│  │  ├─ ModeButtons.jsx
│  │  ├─ ResultBox.jsx
│  │  └─ SubCategoryButtons.jsx
│  ├─ data/
│  │  └─ menus.js        # 메뉴 데이터 (나라, 재료, 설명, 이미지 경로 등)
│  ├─ styles/
│  │  ├─ App.css         # 전체 레이아웃 및 버튼 스타일
│  │  └─ ResultBox.css   # 결과 카드 전용 스타일
│  ├─ App.jsx            # 메인 페이지
│  ├─ main.jsx           # React 진입점
│  └─ index.css
├─ index.html
├─ package.json
└─ vite.config.js
```

## ▶ 실행 방법

### 0. 준비물

- Node.js (최소 18.x 이상 권장)
- npm (Node 설치 시 같이 설치됨)

### 1. 프로젝트 받기

GitHub에서 이 레포지토리를 클론하거나 ZIP으로 다운로드한 뒤,  
터미널/명령 프롬프트에서 프로젝트 폴더로 이동한다.

```bash
git clone <레포지토리_URL>
cd LUNCH_MENU

* ZIP으로 받은 경우에는 압축을 푼 뒤 :
cd 압축_푼_폴더명
```
